# nauto9 手動QAチェックリスト

リリース前／大きな変更後に一通り確認するためのリスト。`npm run lint && npm run typecheck && npm test && npm run build` がすべて通ることを前提に、UI/外部連携の手動確認を行う。

> ⚠️ 本番データ `.dev-data` は壊さないこと。心配な操作はコピーで。

## 0. ビルド・自動チェック
- [ ] `npm run typecheck` エラー0
- [ ] `npm test` 全パス
- [ ] `npm run lint` エラー0
- [ ] `npm run build` 成功

## 1. ウィンドウ / 基本UX
- [ ] ヘッダー上部を掴んでウィンドウを移動できる
- [ ] macOSの信号機（×/−/＋）が押せる・文字と被っていない
- [ ] ホーム画面：ヒーロー画像＋各リンクカードが表示され、クリックで遷移
- [ ] ロゴ／ホームタブでホームに戻れる

## 2. キャラクター
- [ ] 新規キャラ作成 → 一覧に出る
- [ ] 詳細編集：プロンプト／ネガ／**性格・口調(persona)**／メモ／置換 が保存される（onBlur）
- [ ] タグの付与・解除、新規タグ作成
- [ ] **タグ削除**（「タグ編集」→×）：タグが消え、各キャラからも外れる
- [ ] 参照画像のアップロード・並べ替え・参照ON/OFF・グレースケール
- [ ] プレビュー生成（要NovelAIトークン）→ 参照画像として保存
- [ ] 一覧のフリーワード検索・タグ絞り込み

## 3. シチュエーション / ストーリー
- [ ] ストーリー追加・名前変更・削除・並べ替え

- [ ] シチュ追加・編集（名前/プロンプト/ネガ/アスペクト/タグ）
- [ ] **状況メモ / セリフ例**欄が保存される
- [ ] 試し撃ち（プレビュー）→ 背景画像が設定される
- [ ] タグ横断フィルタ・並べ替え

## 4. 一括生成
